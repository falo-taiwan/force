# Force Author Page

這個目錄是 `force.formosa-ai.com` 的作者頁 / 模型菜頁原型。

定位不是主站首頁，而是：

- `Force Cheng` 的個人品牌頁
- 作者首頁＋文章、作品、知識整理與模型菜的入口頁
- 可獨立部署的單頁靜態頁面

## 檔案

- `home-v001.html`: v001 恢復版首頁，主打作者入口頁
- `index.html`: v1.5 首頁，精簡可掃描、最新內容展開、模型菜入口強化
- `menu.html`: v002 模型菜專頁，主打 Menu Experience
- `about_me.html`: 深入認識 Force 的介紹頁
- `ForceAuthorPage.jsx`: 未來版本參考，暫不作為主體維護

## 三版本定位

- v001 首頁：負責讓人認識 Force
- v002 模型菜頁：負責讓人選擇模型菜
- v1.5 首頁：不再折疊，保留精簡重點並導向 `menu.html` 與 `about_me.html`

## 後續可直接接上的內容

- 最新文章連結
- 模型菜詳頁
- 問卷表單
- Email / 聯絡方式
- 外部內容平台入口
