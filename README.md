# Force Author Page

這個目錄是 `force.formosa-ai.com` 的作者頁 / 模型菜頁原型。

定位不是主站首頁，而是：

- `Force Cheng` 的個人品牌頁
- 作者首頁＋文章、作品、知識整理與模型菜的入口頁
- 可獨立部署的單頁靜態頁面

## 版本規則

- `v0xx`
  - 代表前期原型、快速收斂版、演講版或臨時入口版
  - 例如：`v001`、`v002`、`v0.1`

- `v1xx`
  - 代表 `v1.xx`
  - 用於正式主線版本、穩定維護版、小幅升級版
  - 例如：`v101` = `v1.01`

- `v2xx`
  - 代表 `v2.xx`
  - 用於第二代版本、大改版，或不同定位的新版資訊架構
  - 例如：`v201` = `v2.01`

## 檔案

- `home-v001.html`: `v001` 恢復版首頁，屬於 `v0xx` 原型帶
- `index.html`: 目前使用中的首頁，現階段仍屬 `v0xx` / 演講版收斂帶
- `menu.html`: `v002` 模型菜專頁，屬於 `v0xx` 原型帶
- `about_me.html`: 深入認識 Force 的介紹頁
- `ForceAuthorPage.jsx`: 未來版本參考，暫不作為主體維護

## 目前版本定位

- `v001` 首頁：負責讓人認識 Force
- `v002` 模型菜頁：負責讓人選擇模型菜
- 目前 `index.html`：演講 / 入口導流用途的首頁版本

## 未來版本規劃

- `v1xx`
  - 正式作者首頁主線
  - 適合長期維護、穩定部署、對外公開使用

- `v2xx`
  - 第二代作者頁 / 模型菜整合版
  - 適合大幅重整資訊架構、互動方式或內容策略時使用
